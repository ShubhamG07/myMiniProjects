Student Registration System

So in this system Admin can Enter Student details like their Full Name, Email Id, Student Id and their Phone Number and it will be stored in table below the form as soon as he clicks on submit button. Admin can Also edit or delete a student record if it is wrong. Also no empty input or Duplicate Entry Allowed for student id , email or phone number. If same data entered and submitted it will not show in table and also it will show an error Alert with "Student Data Already Exist". Name can only contains alphabtes and space. Mobile No. can Contain Only Numbers with minimum length 10.
